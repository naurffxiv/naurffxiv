# M8S - Howling Blade (Savage)

![Howling Blade](/images/thumbnails/savage/aac_cw/m8s.png)

## Party Finder Summary

```
P1: Fering - Clock - Quad | P2: TF + Rinon
```

## Party Finder Terms

M8S strategies are mix and matched from multiple sources.
All raidplans used for M8S can be found in the [M8S Gigabin](https://pastebin.com/xT2gAw2R).

### Phase 1

#### Fering

Prior to the Millennial Decay mechanic, NA PF follows the [Toxic Friends - M8S P1 Raidplan](https://raidplan.io/plan/46uVU6o49FPuYXOs).
NA PF swaps to the [FERINGBIN M8S Millennial Decay Raidplan](https://raidplan.io/plan/IRl6bjA5WzSweStq)

#### Clock

'Clock' refers to the [Clockspot Terrestrial Rage Raidplan](https://raidplan.io/plan/s3EIvZnk3qvuFy86).

#### Quad

'Quad' refers to the [Quad Moonlight Raidplan](https://raidplan.io/plan/WFsLBku1C9Iyxneu).

### Phase 2

#### TF + Rinon

NA PF follows the [M8S - Toxic/Rinnon Raidplan](https://raidplan.io/plan/Zzf38hNz9aFrn-6T) for the entirety of phase 2.
