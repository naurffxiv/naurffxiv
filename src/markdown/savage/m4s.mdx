
## Party Finder Terms

### "Shabin/Hector"

All strats used are found in [Shaba Pastebin (Shabin) - M4S](<https://pastebin.com/teF90QGm>).
Even if Hector is shown, all strats used are from Shabin.

### "AutoCAD OR Normal Sunrise"

AutoCAD: 
* Cannons are baited using specialized marker edges toward the closest tower. 

Normal: 
* Cannons are baited into the deathwall away from the nearest tower.
